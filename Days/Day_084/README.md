**Day 84: MS Excel Portfolio Project**

**Project Objective:** Engage in an MS Excel portfolio project showcasing proficiency in data manipulation, analysis, and visualization. Use MS Excel to tackle real-world scenarios, such as financial analysis, budgeting, or data tracking.


**Project Tasks:**

Utilize Excel functions, formulas, and data validation for effective data manipulation.
Create charts, pivot tables, and conditional formatting to visualize data trends and insights.
Apply advanced Excel features if applicable, such as Power Query or Power Pivot.
